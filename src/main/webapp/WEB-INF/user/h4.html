用户管理中，是所有关注了微信公众号，注册后的用户信息。如下图：



    用户信息包括：手机号码、姓名、工作单位、所在城市、地区、中标类型等信息。

    级别：用户分四个等级。

    类型：用户类型为：自有员工、运营商电信、运营商联通、运营商移动、合作伙伴、社会能人。用户注册平台后默认为社会用户，区域经理（或选址经理）可点击用户信息后面的“修改用户类型”来修改相关的用户类型信息。


    说明：用户类型和站址对应，社会能人只允许查看四级站址且同时只能抢5个站址，自由员工、运营商、合作伙伴可以查看所有站址且同时抢单数量无限制。


    解冻/冻结：如果某个用户为黑名单用户或其它恶意操作，区域经理（或选址经理）可对用户进行冻结操作。
